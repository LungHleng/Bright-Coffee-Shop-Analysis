# Bright-Coffee-Shop-Analysis
Conducting sales analysis for the CEO to understand the performance of the shop in the last 6 months

### Problem Statement
The Bright Coffee Shop needs data-driven insights on revenue decline, by boosting the revenue and improve the product performance given the historical sales data.

### Aim
IThe aim of the project is to povide insights to the CEO on sales trends, top products, peak hours, and recommendations to drive revenue growth.

### The Objectives
- To identify top revenue-hgenerating products
- To determine peak sales hours
- To analyse sales trends across products and time itervals
- To provide recommendations to improve sales performance

* The following steps were taken to execute the project:
  - Project planning using Canva
  - Brainstorming using Miro
  - Data processing using Databricks

Trends in sales patterns.
Peak sales hour
Top products driving revenue
Data backed recommendations for growth
